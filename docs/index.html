from wisp import WispDevice, capability

class SmartHome(WispDevice):
    device_name = "smart_home"

    @capability
    def read_temperature(self) -> dict:
        """Read the current temperature."""
        sensor = self.hardware.sensor("bme280")
        return {"temperature": sensor.temperature, "unit": "C"}

    @capability
    def set_fan(self, state: str) -> dict:
        """Turn the fan relay on or off."""
        self.hardware.set_output("fan", state == "on")
        return {"fan": state}

device = SmartHome.from_config("config.json")
device.run()  # starts Telegram bot

from wisp import WispDevice
from wisp.plugins.ros2 import ROS2Plugin

class MyRobot(WispDevice):
    device_name = "turtlebot3"

device = MyRobot.from_config("config.json")
device.use(ROS2Plugin())  # scans live ROS2 graph
device.run()            # auto-discovers: movement, lidar, battery…

Simple concept. Powerful result.

You describe what your device can do. WISP tells the AI. The AI translates natural language into a hardware action.

No custom NLP. No training. No server. The AI is grounded in a live list of real capabilities at every request.

  • Capabilities auto-discovered from I2C or ROS2 graph at boot
  • AI returns a structured JSON command every time
  • WISP executes the command on real hardware
  • Result is formatted and sent back to the user
